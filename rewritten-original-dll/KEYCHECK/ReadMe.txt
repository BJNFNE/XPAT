========================================================================
       DYNAMIC LINK LIBRARY : KEYCHECK
========================================================================


Diese KEYCHECK-DLL hat der Anwendungs-Assistent für Sie erstellt.  

Diese Datei enthält eine Zusammenfassung dessen, was Sie in jeder der Dateien
finden, die Ihre KEYCHECK-Anwendung bilden.

KEYCHECK.dsp
    Diese Datei (Projektdatei) enthält Informationen auf Projektebene und wird zur
    Erstellung eines einzelnen Projekts oder Teilprojekts verwendet. Andere Benutzer können
    die Projektdatei (.dsp) gemeinsam nutzen, sollten aber die Makefiles lokal exportieren.

KEYCHECK.cpp
    Dies ist die Hauptquellcodedatei für die DLL.

	Diese DLL exportiert keine Symbole, wenn sie erstellt wird. Daher erzeugt sie 
	bei ihrer Erstellung keine .lib-Datei. Wenn dieses Projekt eine Projektabhängigkeit
	für ein anderes Projekt sein soll, müssen Sie entweder Code zum Exportieren einiger 
	Symbole aus der DLL hinzufügen, damit eine Exportbibliothek erstellt wird, oder Sie 
	können das Kontrollkästchen "Erzeugt keine .LIB" auf der Seite für die Linker- 
	Einstellungen für dieses Projekt aktivieren. 

/////////////////////////////////////////////////////////////////////////////
Weitere Standarddateien:

StdAfx.h, StdAfx.cpp
    Diese Dateien werden zum Erstellen einer vorkompilierten Header-Datei (PCH) namens
    KEYCHECK.pch und einer vorkompilierten Typdatei namens StdAfx.obj verwendet.

Weitere Hinweise:

kennzeichnen, die Sie hinzufügen oder anpassen sollten.
